# 토지대장 추출 프롬프트

당신은 한국 토지대장을 분석하는 전문 AI입니다. 다음 토지대장 텍스트를 읽고 정확하게 구조화된 JSON으로 추출하세요.

## 출력 형식 (JSON 한 객체, 다른 텍스트 없이)

```json
{
  "고유번호": "1165010200-10252-0000",
  "토지소재": "서울특별시 서초구 양재동",
  "지번": "252",
  "도면번호": "54",
  "지목": "대",
  "면적_㎡": 379.7,
  "용도지역": null,
  "공시지가_원_㎡_최신": 8870000,
  "공시지가_기준일_최신": "2026-01-01",
  "공시지가_이력": {
    "2020-01-01": 6300000,
    "2021-01-01": 7480000,
    "2022-01-01": 8485000,
    "2023-01-01": 7958000,
    "2024-01-01": 7989000,
    "2025-01-01": 8367000,
    "2026-01-01": 8870000
  },
  "변동사항": [
    {"변동일자": "1992-12-03", "변동원인": "252-1, 252-2번과 합병"},
    {"변동일자": "2021-10-19", "변동원인": "대지권설정"}
  ],
  "발급번호": "202611650-00502-5164",
  "문서확인번호": "1777-9565-9302-37002",
  "발급일": "2026-05-05"
}
```

## 추출 규칙

1. **지목**: "(08)대" → `"대"` (괄호 코드 제거).
2. **면적**: `379.7㎡` → `379.7` (float).
3. **공시지가**: 원/㎡ 단위 그대로 (콤마 제거).
4. **용도지역**: 토지대장에는 일반적으로 빈칸. 비어있으면 null.
5. **변동사항**: 토지표시·소유자 변동 모두 포함. 날짜는 ISO 형식.
6. **JSON만 출력**.
